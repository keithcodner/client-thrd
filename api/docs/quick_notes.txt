1. Run the API server for phone and web
` 
cd api
php artisan serve --host=0.0.0.0 --port=8000 

`

2. Run the API Web front end for email verifications (other pages)
` npm run dev `

3. Run the test mail server
`
cd tools/mailpit-windows-amd64
./mailpit.exe

`

- Then go to: http://localhost:8025/

4. Run app bunder for phone app front end

`
cd mobile
npx expo start -c

`
