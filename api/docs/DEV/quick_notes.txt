How to install and run Fresh dev instance

Pre-req Steps:
""""""""""""""
Step 1: Install PHP and Composer
For Windows (PHP and Composer install):
1.Install PHP version 8.3+ - from https://www.php.net/downloads.php or download and install XAMPP
2. Download and Install composer from https://getcomposer.org/download/
3. Direct Composer to PHP install folder and php.exe file
4. Check composer version with ` composer -V ` in command prompt window

For macOS (PHP and Composer install):
1. Install Brew with ` /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" `
2. Install PHP version 8.3+ - `brew install php`
3. Install Composer -  `brew install composer`
4. Check composer version with ` composer -V ` in command prompt window


Step 2: Install Node JS
1. Download and Install Node JS (v8.19.2+ or higher) and NPM (v18.12.1+ or higher) from https://nodejs.org/en/download/package-manager

Step 3: Install github
1. Download and Install git from https://git-scm.com/downloads


Step 4: Add installs to environment variables
1. Add each install to your environment variables if necessary via https://www.eukhost.com/kb/how-to-add-to-the-path-on-windows-10-and-windows-11/
2. This is so they can be accessed from any where in your command prompt

Step 5:
1. Download and install mySQL database server https://dev.mysql.com/downloads/mysql/
2. Download mySQL client of your prefrence: ie. https://www.heidisql.com/downloads/releases/HeidiSQL_12.8_64_Portable.zip

Step 6:
1. Install code editor of your choice; prefereably vscode


Install Git repo Steps:
"""""""""""""""""""""""
1. Create a folder on your system for your project ie. ` thrd-react-test `
2. Open this folder in a command prompt window ie. ` cd C:\xampp\htdocs\apps\thrd-react-test `
3. Initiate git; Type the command ` git init `
4. Log into github with required access to https://github.com/thrd/keithcodner. Create Personal access token if required
5. Add remote origin; Type command ` git remote add origin https://github.com/thrd/keithcodner.git `
6. Set url origin with: Type command ` git remote set-url origin https://keithcodner:REPLACE_WITH_PAT@github.com/keithcodner/client-thrd.git
 `
7. Add Upstream: Type command ` git remote add upstream https://github.com/thrd/keithcodner.git `
8. Branch main: Type command ` git branch -M main `
9. Fetch main: Type command ` git fetch ` (it will take time to download)
10. Checkout main: Type command ` git checkout main `
11. The project is now on your system
12. Oopen this project folder in VS Code 

Update .ENV file:
"""""""""""""""""""""""
1. In the root of the project rename the file `.env.exmaple` to `.env`
2. Change the database varable ` DB_DATABASE=laravel ` to ` DB_DATABASE=thrd `
3. Also edit or user name and password for the mysql server you setup

Setup MySQL Database:
"""""""""""""""""""""""
1. Go to your mysql client and create a new empty database called ` thrd `
2. Go to the `database/cor_vers` folder from the project root and import latest .SQL dump file or continue to use migrate command
3. Type command: ` php artisan migrate `
4. Type command: ` php artisan db:seed `


Setup Project for Development:
"""""""""""""""""""""""""""""""
1. Install Composer packages: Type command ` composer install ` from root of project. After a vendor folder should be created
2. Install Nodejs packages: Type command ` npm install ` from root of project. After a node_modules folder should be created


Run Webservers Steps:
"""""""""""""""""""""
1. To start vite for react front end: Type command: ` npm run dev `
1. To start laravel server for back end: Type command: ` php artisan serve `

Laravel/React - How to add new pages:

1. Go to 'resources/js/Pages' folder
2. Create a new folder and a new page ie. Create folder Custom and then create file Welcome_Temp.jsx
3. You will need to add a new route to view the page
4. Go to 'routes/web.php' file
5. Add the following code: (ie. for a new dashboard route) as well as a comment above it for what its for

Route::get('/welcome_temp', [WelcomeController::class, 'index'])->name('welcome_temp');

6. Copy Template_FrontController.php in the directory 'App\Http\Controllers\Front'
7. Rename the file and class name to your desired name (the conroller name must have 'Controller' suffix in the naming convention ie. ReservationController for Reservation page)
8. In the controller point the render() function, it references the 'resources/js/Pages' where the the Pages directory acts as the root

public function index()
{
    return Inertia::render('Custom/Welcome_Temp', []);
}
9. View the page by going to the route and see if it works
